<h3 align="left">VT-Express</h3>

<p align="left">
  Это десктопное приложение, предназначенное для оперативной проверки подозрительных файлов на наличие вредоносного ПО. Инструмент объединяет в себе мощь облачных антивирусных лабораторий <b>VirusTotal</b> и возможности <b>искусственного интеллекта</b> для глубокой интерпретации результатов.
